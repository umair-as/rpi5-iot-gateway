About Thread

Thread is a low-power, secure, IPv6-based mesh networking protocol designed for IoT devices. It provides:

  • Reliability — Self-healing mesh with no single point of failure
  • Security — Bank-grade encryption with secure commissioning
  • Low Power — Battery-operated devices can run for years
  • IP-based — Native IPv6 connectivity to cloud services
