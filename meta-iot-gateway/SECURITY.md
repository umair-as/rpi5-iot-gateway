# Security Policy

Security policy, hardening posture, and threat model for this layer are
maintained at the project level: see
[`docs/SECURITY.md`](../docs/SECURITY.md) in the repository root.

To report a vulnerability, use GitHub's private vulnerability reporting on
[umair-as/rpi5-iot-gateway](https://github.com/umair-as/rpi5-iot-gateway/security)
rather than a public issue.
